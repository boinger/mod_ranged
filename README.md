mod_ranged
==========
Apache module for [libcrange](https://github.com/boinger/libcrange)

## Prereqs
### To build
    apr-util-devel
    httpd-devel
    pcre-devel
    sqlite-devel
    zlib-devel

### To run
    flex
    libyaml
    perl-ExtUtils-MakeMaker
    perl-ExtUtils-Embed
    perl-Test-Simple
    perl-libwww-perl
    pcre
    zlib
